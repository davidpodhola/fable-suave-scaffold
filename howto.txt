I set framework: >= net461 in the paket.dependencies file
I set the same value in <TargetFramework>net461</TargetFramework>
I added <RuntimeIdentifier>win7-x64</RuntimeIdentifier> under the TargetFramework value

you can then run:

david.podhola@DESKTOP-3EFJ3F0 C:\Users\david.podhola\Documents\S\fable-suave-scaffold\src\Server\bin\Debug\net461
> Server.exe C:\Users\david.podhola\Documents\S\fable-suave-scaffold\src\Client

or upload C:\Users\david.podhola\Documents\S\fable-suave-scaffold\src\Server\bin\Debug\net461 with client subfolder

and then web.config like

<?xml version="1.0" encoding="UTF-8"?>
<configuration>
    <system.webServer>
        <handlers>
            <remove name="httpplatformhandler" />
            <add name="httpplatformhandler"
                path="*"
                verb="*"
                modules="httpPlatformHandler"
                resourceType="Unspecified" />
        </handlers>
        <httpPlatform processPath="./Server.exe"
            arguments=""
            stdoutLogEnabled="true"
            stdoutLogFile="./Server.log"
            startupTimeLimit="20"
            requestTimeout="00:05:00">
        </httpPlatform>
    </system.webServer>
</configuration>
